# X2 Financial Game (Flutter copy)

Это Flutter-версия приложения из текущего репозитория, вынесенная в отдельную подпапку.

## Что реализовано
- Главный экран с текущим днем, лимитом и прогрессом до максимума.
- Экран трат дня с проверкой превышения лимита (+5% буфер).
- История дней.
- Ачивки/streak и депремирование после двух пропусков.
- Настройки языка (ru/en/de), валюты/лимитов, поведения на максимуме.
- Локализация названия: **X2 Financial Game** + переводы.

## Запуск
1. Установить Flutter SDK.
2. В этой папке выполнить:
   - `flutter pub get`
   - `flutter run`

## Тесты
- `flutter test`
